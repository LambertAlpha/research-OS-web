# Folder: components

1. **地位**: UI 组件层。存放所有可复用的 React 展示组件，负责纯 UI 渲染，不包含业务状态管理。
2. **逻辑**: 接收由页面组件 (app/) 传入的 props（数据、回调函数）→ 渲染对应的 UI 结构 → 通过回调通知父组件用户操作。
3. **约束**: 组件必须是纯展示型，不直接调用 API；状态管理仅限 UI 级别（如输入框值、加载态）；所有组件均标记为 `"use client"`。

## 成员清单

- `Sidebar.tsx`: 左侧导航栏 (Navigation) — 固定布局，路由链接 + 系统状态指示
- `Header.tsx`: 顶部工具栏 (Toolbar) — 日期选择器 + 运行模型按钮 + 最后更新时间
- `MetricCard.tsx`: 指标卡片 (Data Display) — 单个数值指标的卡片式展示
- `GateCard.tsx`: 闸门卡片 (Data Display) — 门控状态可视化，含进度条和状态指示器
- `Chart.tsx`: 通用图表 (Data Visualization) — 基于 Recharts，支持面积图/折线图 + 参考线
- `providers.tsx`: 全局 Provider (Context Wrapper) — React Query 客户端配置与注入
