# Folder: types

1. **地位**: 契约层。定义前后端之间所有数据结构的 TypeScript 类型，是数据契约的唯一真相源 (Single Source of Truth)。
2. **逻辑**: 类型定义与后端 Python 模型的输出结构一一对应（ModelOutput ↔ ModelOutputResponse v2.0），被 `lib/api.ts` 和所有页面/组件引用。
3. **约束**: 仅包含 `interface` / `type` 定义，不包含运行时代码；任何后端响应结构变更必须同步更新此目录下的类型。

## 成员清单

- `api.ts`: API 类型定义全集 (Type Contracts) — 涵盖 LiquidityOutput v3.0、MacroOutput v4.0、ModelOutput v2.0、市场数据类型

**触发器**: 一旦本文件夹增删文件或架构逻辑调整，必须重写此文档。
