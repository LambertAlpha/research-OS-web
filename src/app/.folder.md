# Folder: app

1. **地位**: 页面路由层。基于 Next.js App Router 约定式路由，定义用户可达的所有 URL 及其对应的页面组件。
2. **逻辑**: 每个子目录对应一个路由路径；页面组件内管理局部状态（useState），调用 `lib/api.ts` 获取数据，将结果传递给 `components/` 层渲染。
3. **约束**: 页面组件只做状态编排与数据流转，不包含可复用 UI 逻辑（属于 components）；不包含 HTTP 请求逻辑（属于 lib/api.ts）。

## 成员清单

- `layout.tsx`: 全局根布局 (Root Layout) — 注入 Sidebar、Providers、全局样式
- `page.tsx`: 首页 — 市场概览仪表板 (/)
- `globals.css`: 全局样式 — CSS 变量、自定义组件类、动画定义
- `liquidity/`: 流动性模型页面 (/liquidity)
- `macro/`: 宏观模型页面 (/macro)
- `history/`: 运行历史页面 (/history)

**触发器**: 一旦本文件夹增删路由或页面逻辑调整，必须重写此文档。
