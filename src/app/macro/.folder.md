# Folder: macro

1. **地位**: 宏观模型 v4.0 专属页面。多维度展示市场信号分析：利率结构(L1)、叙事校验(L2)、风险闸门(L3)、执行矩阵(L4)。
2. **逻辑**: 接收用户"运行模型"操作 → 调用 `apiClient.runModel()` → 解析 `ModelOutput.macro` (MacroOutput) → 并发获取 DXY/HY_OAS/IG_OAS/VIX/SPX/US10Y 市场数据 → 分层渲染四个 Layer 区块。
3. **约束**: 仅关注宏观维度数据，不渲染流动性模型内容；宏观状态颜色映射(A/B/C/D)在页面内定义。
