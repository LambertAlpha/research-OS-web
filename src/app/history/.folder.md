# Folder: history

1. **地位**: 运行历史页面。展示模型单次运行的触发规则、告警信息和运行元数据。
2. **逻辑**: 接收用户"运行模型"操作 → 调用 `apiClient.runModel()` → 解析 `ModelOutput.triggered_rules` 和 `ModelOutput.alerts` → 渲染规则表格、告警列表和运行信息摘要。
3. **约束**: 当前版本为单次运行结果展示，历史记录功能依赖后端数据库支持（尚未实现）。
